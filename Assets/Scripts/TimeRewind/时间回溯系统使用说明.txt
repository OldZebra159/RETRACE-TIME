================================================================================
                           时间回溯系统 - 完整方案文档
================================================================================

【概述】
本时间回溯系统完全基于你现有的RPG项目架构设计，无需修改任何原有代码，
通过添加新的组件和脚本即可实现完整的时间回溯功能。

================================================================================
【核心组件说明】

1. TimeRecordData.cs - 时间数据记录类
   - 功能：存储每一帧的实体状态数据
   - 记录内容：位置、速度、朝向、生命值、动画状态、异常状态等

2. ITimeRewindable.cs - 可回溯实体接口
   - 功能：定义时间回溯所需的基本方法
   - 方法：RecordState()、RestoreState()、GetRecordHistory()、ClearHistory()

3. TimeRewindManager.cs - 时间回溯管理器（单例）
   - 功能：全局管理所有可回溯对象
   - 特性：
     * 默认按键：R键（按住回溯，松开停止）
     * 默认记录时长：5秒
     * 回溯速度：2倍速
   - 主要方法：
     * RegisterRewindable() - 注册可回溯对象
     * UnregisterRewindable() - 注销可回溯对象
     * StartRewind() - 开始回溯
     * StopRewind() - 停止回溯

4. PlayerTimeRewind.cs - 玩家时间回溯组件
   - 功能：为玩家提供时间回溯能力
   - 使用方法：直接添加到Player游戏对象上

5. EnemyTimeRewind.cs - 敌人时间回溯组件
   - 功能：为敌人提供时间回溯能力（包括复活功能）
   - 使用方法：直接添加到Enemy游戏对象上

6. TimeRewind_Skill.cs - 时间回溯技能
   - 功能：将时间回溯作为可解锁技能集成到SkillManager中
   - 特点：有冷却时间、可解锁

7. TimeRewindFX.cs - 时间回溯特效组件
   - 功能：提供视觉和听觉特效
   - 可选组件：用于增强回溯时的视觉体验

================================================================================
【快速部署步骤】

步骤1：创建TimeRewindManager对象
   - 在场景中创建一个空GameObject，命名为"TimeRewindManager"
   - 添加TimeRewindManager.cs脚本组件
   - 配置参数：
     * Record Duration：记录时长（秒）
     * Rewind Speed：回溯速度
     * Rewind Key：回溯按键

步骤2：为玩家添加时间回溯能力
   - 找到Player游戏对象
   - 添加PlayerTimeRewind.cs组件
   - （可选）添加TimeRewindFX.cs组件用于特效

步骤3：为敌人添加时间回溯能力
   - 找到所有Enemy预制体或场景中的Enemy对象
   - 添加EnemyTimeRewind.cs组件
   - （可选）添加TimeRewindFX.cs组件

步骤4：（可选）集成到技能系统
   - 在SkillManager游戏对象上添加TimeRewind_Skill.cs组件
   - （注意：如需完全集成到SkillManager，需要在SkillManager.cs中添加引用）

【功能特点】

✓ 完全无侵入 - 不修改任何原有代码
✓ 易于扩展 - 可轻松为其他对象添加时间回溯能力
✓ 完整状态恢复 - 包括位置、生命值、异常状态等
✓ 敌人复活 - 回溯时可复活已死亡的敌人
✓ 可配置参数 - 记录时长、回溯速度、按键均可配置
✓ 性能优化 - FixedUpdate记录，限制最大记录数

【高级扩展建议】

1. 集成到技能树
   - 将TimeRewind_Skill添加到技能树UI
   - 通过技能点解锁时间回溯能力

2. 自定义特效
   - 创建专门的回溯材质（Shader）
   - 添加粒子系统效果
   - 使用项目中已有的时钟音效（sfx_chronosphere.mp3）

3. 记录更多状态
   - 可扩展TimeRecordData记录更多信息
   - 例如：技能冷却、物品状态等

4. 回溯视觉轨迹
   - 显示回溯路径的残影效果
   - 参考AfterImageFX的实现

5. 项目物回溯
   - 为投掷物（剑、箭矢等）添加回溯组件
   - 实现"时间倒流"的视觉效果

================================================================================
【注意事项】

1. 性能考虑：
   - 记录时长建议控制在3-10秒
   - 过多的可回溯对象会增加内存占用
   - 可考虑对敌人使用对象池优化

2. 音效：
   - TimeRewindManager中使用了AudioManager.instance.PlaySFX(30)
   - 请根据实际音效索引调整或删除

3. 状态恢复：
   - 当前实现恢复了主要状态，可根据需要扩展
   - 动画状态恢复可进一步优化
