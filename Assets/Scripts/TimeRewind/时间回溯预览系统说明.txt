================================================================================
                        时间回溯预览系统 - 使用说明
================================================================================

【系统概述】
本系统在原有时间回溯功能的基础上，添加了时间点预览和选择功能，
让玩家可以在回溯时看到不同时间点的角色状态，并选择恢复到特定时间点。

================================================================================
【核心功能】

1. 时间点预览
   - 按下R键时进入预览模式
   - 显示过去3秒内每0.5秒的角色状态
   - 以半透明的方式展示不同时间点的角色位置和状态

2. 时间点选择
   - 鼠标点击选择特定时间点
   - 左右方向键切换选择
   - 空格键确认选择并恢复到该时间点

3. 视觉效果
   - 预览时游戏进入慢动作
   - 选中的时间点会高亮显示
   - 清晰展示角色的移动轨迹和状态变化

================================================================================
【部署步骤】

**步骤1**：为玩家添加组件
   - 添加 `PlayerTimeRewind.cs` （已存在）
   - 添加 `TimeRewindPreview.cs` （新增）

**步骤2**：配置预览参数
   - maxPreviewCount = 6 （默认）
   - previewDuration = 3 （默认）
   - previewInterval = 0.5 （默认）
   - 其他参数可根据需要调整

**步骤3**：测试操作
   - 按 **R键** 进入预览模式
   - 点击或按方向键选择时间点
   - 按 **空格键** 确认恢复

================================================================================
【操作说明】

| 按键 | 功能 |
|------|------|
| R键 | 进入/退出预览模式 |
| 鼠标左键 | 点击选择时间点 |
| 左右方向键 | 切换选择时间点 |
| 空格键 | 确认选择并恢复 |

================================================================================
【配置选项】

1. **TimeRewindPreview.cs**：
   - maxTrailCount：最大预览数量
   - previewDuration：预览总时长
   - previewInterval：预览间隔
   - previewColor：预览颜色
   - selectedPreviewColor：选中预览颜色

2. **TimeRewindManager.cs**：
   - confirmKey：确认键（默认Space）
   - rewindKey：回溯键（默认R）

================================================================================
【技术特点】

1. **无侵入性**：完全基于现有系统，不修改原有代码
2. **性能优化**：智能管理预览对象，避免内存泄漏
3. **用户友好**：直观的视觉界面和操作方式
4. **可扩展性**：可轻松调整预览数量和间隔

================================================================================
【使用场景】

- **精准回溯**：精确恢复到特定时间点，避免过度回溯
- **教学演示**：展示角色的移动轨迹和操作过程
- **策略规划**：观察过去的操作，制定更好的策略
- **视觉效果**：增强游戏的视觉体验和科技感

================================================================================
